# 📚 Bookspy

Bookspy is a simple JavaScript web application to manage books.  
You can add and delete books using a popup form interface.

---

## 🚀 Features
- Add book title, author, and description
- Delete books dynamically
- Popup form UI
- Simple and clean design

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

---

## 📸 How it works
1. Click the **+ button**
2. Enter book details
3. Click **ADD**
4. Book appears in list
5. Click **Delete** to remove book
